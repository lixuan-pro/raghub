# RAGHub 论文材料占位

## 1. 当前状态

本目录用于保存 RAGHub 项目后续与毕业论文相关的材料。

当前项目仍处于 Week 1 工程骨架收口阶段，论文内容暂不正式展开。本文件只作为论文线的最小占位，方便后续逐步补充学校论文要求、题目候选、技术决策和实验设计草稿。

## 2. 后续计划存放内容

后续本目录主要用于保存以下内容：

1. 学校毕业论文要求与时间节点
2. 论文题目候选
3. 项目相关技术决策记录
4. 实验设计草稿
5. 评测指标与实验结果整理
6. 可写入论文的系统设计材料

## 3. 当前不做的内容

当前阶段暂不展开以下内容：

- 不确定正式论文题目
- 不撰写论文正文
- 不展开完整实验设计
- 不整理参考文献
- 不提前写系统架构章节

## 4. 当前与项目进度的关系

RAGHub 当前仍处于基础设施阶段，已经完成 FastAPI 最小服务、配置管理、日志模块和基础测试。

后续进入文档导入、切块、检索、生成问答和评测阶段后，再逐步把技术决策、实验设计和结果分析沉淀到本目录中。
